# Forensic Quiz

This repository contains an example iOS application called **ForensicQuizzer**. The app lets you practice forensic science quizzes on your iPhone and learn from detailed explanations.

## Features
- Subject categories including Toxicology, Entomology and Serology
- Multiple choice questions with instant feedback
- Explanation of each answer
- Built with SwiftUI

## Usage
Open the `ForensicQuizzerApp` folder in Xcode and run on a simulator or device. Sample questions are provided in `SampleData.swift`.
